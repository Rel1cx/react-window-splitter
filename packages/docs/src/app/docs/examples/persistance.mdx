# Persistance

You can persist the state of the layout by using the `autosaveId` prop on the `PanelGroup`.
You muse also provide a unique id for each panel.

<div className="my-6">
  <PersistanceExampleVisual />
</div>

```tsx
import { PanelGroup, Panel, PanelResizer } from "react-window-splitter";

function PersistanceExample() {
  return (
    <PanelGroup autosaveId="autosave" style={{ height: 200 }}>
      <Panel id="panel-1" min="130px" max="400px" />
      <PanelResizer id="resizer-1" />
      <Panel id="panel-2" min="130px" />
    </PanelGroup>
  );
}
```

## Cookie Storage

By default the layout will be persisted to `localStorage`.

If you want to persist to `cookie` you can set `autosaveStrategy='cookie'`.
